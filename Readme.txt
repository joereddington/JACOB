# URL arguments

1. url

   ?url=https%3A%2F%2Fexample.com

2. seconds

   ?seconds=45

3. message

   ?message=Thank%20you

4. effect

   - shrink
   - fade

   Example:
   ?effect=fade

## Defaults

- url = https://example.com
- seconds = 30
- message = Thank you!
- effect = shrink
